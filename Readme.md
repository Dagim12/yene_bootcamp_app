# HaHu Bootcamp API

> Backend API for HaHu application, which is a bootcamp directory website

## Usage

Rename congif/config.env.env to config/config.env and update the values/setting to your own

## Install dependencies

```
npm install
```

## Run App

```
npm run dev
```

## Run in prod

```
npm start
```
